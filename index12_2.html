Плавна трансформація CSS властивість transition -- transform

Псевдоклас :hover дозволяє елементам бути менш статичними, змінюючи їх властивості при наведенні мишки.

при наведенні мишки
<style>
.blok {
  background: #808991;
  color: #FFF;
  cursor: pointer;
}
.blok:hover {
  background: #D0D941;
  color: #000;
}
</style>

<div class="blok">при наведении мышки</div>

Завдяки властивості transition (w3.org) можна зробити плавний перехід між станами елемента.

плавна зміна кольору туди-назад
<style>
.transition {
  transition: 3s;
}
</style>

<div class="blok transition">плавна зміна кольору туди-назад</div>
плавна зміна при наведенні на div
<style>
.transition2:hover {
  transition: 3s;
}
</style>

<div class="blok transition2">плавна зміна при наведенні на div</div>
плавна зміна, коли курсор виведений за межі div
<style>
.transition3:not(:hover) {
  transition: 3s;
}
</style>

<div class="blok transition3">плавна зміна, коли курсор виведений за межі div</div>
збереження становища :hover
<style>
.transition4 {
  transition: 0s 9999999s;
}
.transition4:hover {
  transition: 3s;
}
</style>

<div class="blok transition4">збереження становища :hover</div>
Властивість Опис
transition-property властивість [список можливих], на яке поширюється трансформація. При необхідності на кілька властивостей, вони перераховуються через кому.

  • none — ніхто

transition-timing-function вказує, як повинні змінюватися проміжні значення в кривій часу. Наприклад, починати розвиватись повільно, а потім прискорюватися. Трансформація може відбуватися і плавно протягом усієї ділянки часу [cubic-bezier], і ривками в задану кількість кроків [steps].
  • ease, он же cubic-bezier(0.25, 0.1, 0.25, 1.0)
  • linear, он же cubic-bezier(0.0, 0.0, 1.0, 1.0)
  • ease-in, он же cubic-bezier(0.42, 0, 1.0, 1.0)
  • ease-out, он же cubic-bezier(0, 0, 0.58, 1.0)
  • ease-in-out, он же cubic-bezier(0.42, 0, 0.58, 1.0)
  • step-start, он же steps(1, start)
  • step-end, он же steps(1, end), он же steps(1)

плавна зміна кольору CSS
<style>
.transition1 {
  transition: background-color 3s cubic-bezier(0.1, 0.8, 0.5, 2), color 3s 1s linear;

  /* або */

  transition-property: background-color, color;
  transition-duration: 3s;
  transition-delay: 0s, 1s;
  transition-timing-function: cubic-bezier(0.1, 0.8, 0.5, 2), linear;
}
</style>

<div class="blok transition1">плавна зміна кольору CSS</div>
